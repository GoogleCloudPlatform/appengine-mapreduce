MapReduce Sample Programs

Random collisions example
This example demonstrates parallel computation. It looks for collisions in Java's random number generator,
where a collision is defined as multiple seed values that produce the same output value when next() is called.
The input source is a range of numbers to test. Collisions are logged and written out to a file in Google Cloud Storage.
(No collisions occur so the file will be empty.)
Run the example.
